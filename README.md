# 🚀 ResumePilot

### Navigate Your Career Journey

ResumePilot is an AI-powered resume analysis and job matching platform designed to help job seekers improve their resumes and evaluate their suitability for specific roles.

The application analyzes uploaded resumes, extracts relevant information, identifies technical skills, calculates an ATS compatibility score, compares resumes against job descriptions, checks essential resume sections, provides personalized recommendations, and generates a downloadable PDF report.

---

# ✨ Features

- 📄 Upload Resume (PDF)
- 📋 Resume Text Extraction
- 🎯 ATS Compatibility Score
- ✅ Matched Skills Detection
- ❌ Missing Skills Analysis
- 📑 Resume Section Checker
- 🤖 AI-Powered Recommendation
- 📄 Downloadable PDF Report
- 🎨 Clean and Responsive User Interface

---

# 📊 How It Works

1. Upload your resume in PDF format.
2. Paste the target job description.
3. ResumePilot extracts text from the resume.
4. Skills are compared with the job description.
5. An ATS compatibility score is calculated.
6. Matched and missing skills are identified.
7. Resume sections are analyzed.
8. A personalized AI recommendation is generated.
9. Download the complete analysis as a PDF report.

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/bhavyasrivastava5april/ResumePilot.git
```

### Navigate to the project folder

```bash
cd ResumePilot
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open in your browser

```text
http://127.0.0.1:5000
```
